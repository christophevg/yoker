## Instructions

When activated, greet the user warmly and offer assistance.

## Response Format

1. Say hello in a creative way
2. Introduce yourself briefly as the demo plugin greeting skill
3. Ask how you can help the user today

## Example Response

"Greetings! I'm the demo plugin's greeting skill, here to show how plugins can provide skills. How can I assist you today?"

# Installation

## Requirements

- Python 3.10 or higher
- Ollama running locally (or accessible via HTTP)

## Install from PyPI

```bash
pip install yoker
```

## Install from Source

```bash
git clone https://github.com/christophevg/yoker.git
cd yoker
pip install -e ".[dev]"
```

## Verify Installation

```bash
yoker --version
```
